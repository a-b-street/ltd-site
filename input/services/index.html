A/B Street Ltd is available for building bespoke software from scratch and for adapting existing projects to fit new datasets and use cases. We can take on any project, but we prefer open source projects with a focus on sustainable transportation and land use, and involving co-design between a community and government. We have lots of experience working with transportation domain experts in government, industry, and academia.

Particular specialties include:

  • traffic simulation and agent based modelling
  • interactive data visualization and storytelling
  • GIS and OpenStreetMap
  • routing
  • designing tools both for professional and general audiences

A/B Street Ltd has specialized in a "no server needed" tech stack, building deeply interactive applications that run entirely in a web browser, leveraging the power of WebAssembly and serverless file formats.

About the team

A/B Street Ltd is a one-person consultancy run by Dustin Carlino, who has been building open source transportation planning software since 2018. In 2020 he launched the alpha of A/B Street (the company's namesake), a gamified traffic simulation rendering OpenStreetMap data in unprecedented detail and running entirely in a web browser. He spent the next year prototyping more targeted tools like Ungap the Map and the 15-minute neighbourhood tool. He joined the Alan Turing Institute in 2022, built od2net and the first version of the LTN tool, then joined Active Travel England on secondment for two years, building Plan Your Active Travel Schemes and Route Check. In early 2025, he formed A/B Street Ltd to provide full-time software consultancy services, where he has built two tools for Sustrans, including the Network Planning Workspace.
